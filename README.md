# Budget Book / 家計簿

A lightweight personal budget tracking app built with React + TypeScript.

## Features

- **Dashboard** - Overview of assets, liabilities, income, expenses and net worth
- **Transaction Management** - Add, edit, delete entries with category filtering
- **Subscription Tracker** - Manage monthly subscriptions
- **Asset History** - Track asset trends over time with interactive charts
- **Data Backup** - Export/import data as JSON files
- **Bilingual** - Japanese / English language support
- **Offline First** - All data stored locally in your browser (localStorage)

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS 4
- Chart.js

## Getting Started

```bash
npm install
npm run dev
```
